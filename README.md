##### Overview
###### This project focuses on analyzing e-commerce user behavior to understand the factors that influence purchasing decisions. The dataset contains user interactions such as product views, cart additions, and purchases. The goal of this analysis is to identify patterns in user activity and provide insights that can improve customer conversion and business decision-making.

###### The project combines data cleaning, exploratory analysis, machine learning, and data visualization to create a complete end-to-end analytical solution. By transforming raw data into meaningful insights, this project demonstrates how data can be used to support real-world business strategies.

##### Objectives
###### Analyze user behavior across different interaction stages (view, cart, purchase)
Identify key factors influencing purchase decisions
Build a predictive model to estimate purchase likelihood
Develop an interactive dashboard for business insights
##### Data Preparation

###### The dataset was cleaned and preprocessed using Python. Missing values and duplicate records were handled to ensure data quality. Data types were corrected, and new features were created to better represent user activity and engagement patterns.

###### Additional transformations included converting timestamps into usable formats and generating a binary target variable to indicate whether a purchase occurred. These steps improved the dataset’s suitability for analysis and modeling.

##### Machine Learning

###### A logistic regression model was implemented to predict whether a user is likely to make a purchase based on their activity and behavior. Feature engineering was applied to capture meaningful patterns, such as user engagement levels and average spending behavior.

###### The model helps in understanding how different variables influence purchase decisions and provides a foundation for customer targeting and retention strategies.

##### Dashboard (Power BI)

###### An interactive Power BI dashboard was created to visualize key insights and support decision-making. The dashboard includes:

###### Key performance indicators (KPIs) such as total users, purchases, and conversion rate
A conversion funnel showing the user journey from viewing to purchasing
Trend analysis of user activity over time
Category and product-level performance insights
Interactive slicers for filtering by category, brand, and time

###### The dashboard is designed to be user-friendly and allows stakeholders to explore data dynamically.

##### Key Insights
###### Users with higher activity levels are more likely to make purchases
A small number of categories contribute to the majority of sales
Conversion rate is a strong indicator of user engagement
Pricing and product selection influence purchase behavior

##### Tools & Technologies
###### Python (Pandas, NumPy, Scikit-learn)
###### Power BI (Dashboard & Data Visualization)
###### Excel (Data handling and initial exploration)
 
##### Conclusion
###### This project demonstrates how data analysis, machine learning, and visualization can be combined to generate actionable business insights. By understanding user behavior and identifying key drivers of conversion, businesses can make more informed decisions to improve performance and customer experience.
